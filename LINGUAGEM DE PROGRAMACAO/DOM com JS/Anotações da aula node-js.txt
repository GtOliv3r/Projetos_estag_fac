LINHA DO TEMPO JAVASCRIPT

1970

Guerra Fria, existia um embate tecnológico entre EUA e RUSSIA. Onde a RUSSIA estava na frente por varios motivos, como lamcamento de satelite.
EUA fundou o DARPA. Para não haver perda de dados em caso de ataque na base, eles usaram a ARPANET.

1993
Com a fama da rede criada para salvamento de dados em backup, um cara foi la e criou linguagem de marcação HTML, HTTP.
Criado tbm o MOSAIC, primeiro navegador. o www também.

1994

Novo navegador chamado NeTSCAPE, resolvia varios bugs que existiam no MOSAIC, tornando-se o navegador mais utilizado. tbm criaram o modzilla

1995

Fundação da Microsoft, e ela criou o Internet Explorer. Oracle estava focando na criação da linguagem de programação JAVA.
NETSCAPE também estava criando uma outra linguagem de programação, chamada MOCA, depois mudando para LIVESCRIPT, que depois mudou para JAVASCRIPT


1997

Padronização da linguagem basica JAVASCRIPT pela empresa ECMA, surgindo assim o ECMASCRIPT.


2002

NETSCAPE faliu pois Microsoft estava detonando geral. Porém uma das equipes do NETSCAPE não abandonou todos os projetos, aproveitando o Modzilla e criando o Mozila e depois o Firefox.

2008
Google cria o navegador Chrome, em 3 meses se torna o primeiro mundial. 

2009

Criando o V8, fazendo alusão ao motores V8


2010

Node.js é um software de código aberto, multiplataforma, baseado no interpretador V8 do Google 
e que permite a execução de códigos JavaScript fora de um navegador web.
A principal característica do Node.js é sua arquitetura assíncrona e orientada por eventos.
